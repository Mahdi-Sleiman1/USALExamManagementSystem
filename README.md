# USAL Exam Management System

A web-based exam management system developed using ASP.NET Core and Entity Framework.

## Roles
- Admin
- Doctor
- Student

## Features (Current Progress)
- User authentication (Login / Register)
- Role-based dashboards
- Database with Majors, Courses, Exams
- Doctor can create exams
- Student can view exams

## Technologies
- ASP.NET Core
- Razor Pages
- Entity Framework Core
- SQL Server
- Bootstrap
- Rest API
